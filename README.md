# koa2-template-lite

基于 koa2 + typescript 的简单服务器模板，快速搭建 mock server

## http 服务

```bash
yarn
yarn dev # 热更模式
# yarn start
```

### 接口调试

浏览器调试或者 postman

## websocket 服务

```bash
yarn
yarn dev:ws
# yarn start:ws
```

### 接口调试

调试网站： http://www.websocket-test.com/
